MPFR Third-Party Packages and Ports

MPFR is also available via third-party packages and ports:

  • Official Debian packages libmpfr-dev, libmpfr1ldbl (shared library) and source package.
  • Gentoo MPFR packages.
  • Arch Linux MPFR package.
  • FreeBSD mpfr port (FreshPorts).
  • NetBSD mpfr port (pkgsrc).
  • MacPorts MPFR port (for Mac OS X).
  • Fink libmpfr1 package.
  • OpenPKG mpfr package.
  • libmpfr for Solaris 8 and later.
  • HP-UX MPFR packages.
  • MPFR for Microsoft Windows, using Microsoft Visual Studio 2008.
  • DJGPP port of MPFR 2.3.1.
  • Cygwin MPFR packages.
  • MinGW MPFR ports.
  • OpenVMS port of MPFR.
