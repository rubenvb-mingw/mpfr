GNU MPFR Third-Party Packages and Ports

GNU MPFR is also available via third-party packages and ports:

  • Official Debian packages: libmpfr-dev (development files) and libmpfr-doc (documentation). More information: Debian source package.
  • Ubuntu MPFR packages.
  • Gentoo MPFR packages.
  • Arch Linux MPFR package.
  • FreeBSD mpfr port (FreshPorts).
  • NetBSD mpfr port (pkgsrc).
  • MacPorts MPFR port (for Mac OS X).
  • Fink libmpfr4 and Fink libmpfr4-64bit packages.
  • HP-UX MPFR packages.
  • MPFR for Microsoft Windows, using Microsoft Visual Studio 2008.
  • DJGPP MPFR packages.
  • Cygwin MPFR packages.
  • MinGW MPFR ports.
  • OpenVMS port of MPFR.
